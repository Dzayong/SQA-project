# Black Box Testing

Black Box Testing adalah metode pengujian software yang berfokus pada fungsi sistem berdasarkan input dan output tanpa melihat kode program.

## Tujuan
Untuk memastikan sistem berjalan sesuai dengan kebutuhan dan harapan pengguna.

## Karakteristik
- Tidak melihat source code
- Berfokus pada fungsi aplikasi
- Berdasarkan kebutuhan user

## Teknik
- Equivalence Partitioning
- Boundary Value Analysis
- Decision Table Testing
- State Transition Testing
- Use Case Testing

## Inti
"Apakah sistem bekerja sesuai yang diharapkan user?"
