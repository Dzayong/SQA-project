# Branch Coverage

## Pengertian
Branch Coverage adalah teknik pengujian yang memastikan setiap percabangan (if-else) dalam program diuji.

## Tujuan
Untuk memastikan semua kemungkinan hasil dari kondisi telah dijalankan.

## Contoh
Jika ada kondisi:
if (x > 10)
- Harus diuji saat kondisi TRUE dan FALSE

## Kelebihan
- Lebih lengkap dibanding statement coverage
- Menguji logika percabangan

## Kekurangan
- Tidak mencakup semua jalur kombinasi

## Kesimpulan
Branch Coverage memastikan setiap keputusan dalam kode telah diuji.
