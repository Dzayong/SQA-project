# Loop Testing

## Pengertian
Loop Testing adalah teknik pengujian yang berfokus pada struktur perulangan seperti for, while, dan do-while.

## Tujuan
Untuk memastikan perulangan berjalan dengan benar dalam berbagai kondisi.

## Contoh
- Loop tidak dijalankan (0 kali)
- Loop dijalankan 1 kali
- Loop dijalankan maksimal

## Kelebihan
- Fokus pada bagian yang sering error (loop)

## Kekurangan
- Tidak mencakup seluruh program

## Kesimpulan
Loop Testing memastikan perulangan tidak menyebabkan error atau infinite loop.
