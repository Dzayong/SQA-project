# Control Flow Testing

## Pengertian
Control Flow Testing adalah teknik pengujian yang menganalisis alur kontrol program berdasarkan struktur logika.

## Tujuan
Untuk memastikan alur eksekusi program berjalan sesuai desain.

## Contoh
Menggunakan diagram alur (flowchart) untuk melihat jalur program.

## Kelebihan
- Membantu memahami alur program secara keseluruhan

## Kekurangan
- Membutuhkan pemahaman logika yang cukup tinggi

## Kesimpulan
Control Flow Testing memastikan alur logika program berjalan dengan benar dan terstruktur.
