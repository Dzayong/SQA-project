# Path Coverage

## Pengertian
Path Coverage adalah teknik pengujian yang memastikan semua jalur kemungkinan eksekusi dalam program diuji.

## Tujuan
Untuk menguji seluruh kombinasi alur program.

## Contoh
Program dengan beberapa kondisi akan memiliki banyak jalur yang harus diuji satu per satu.

## Kelebihan
- Sangat lengkap dalam menguji logika program

## Kekurangan
- Kompleks dan memakan waktu
- Tidak efisien untuk program besar

## Kesimpulan
Path Coverage memberikan pengujian paling menyeluruh terhadap alur program.
