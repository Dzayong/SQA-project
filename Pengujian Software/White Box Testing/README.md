# White Box Testing

White Box Testing adalah metode pengujian yang berfokus pada struktur internal kode program.

## Tujuan
Memastikan seluruh logika dalam program berjalan dengan benar.

## Teknik
- Statement Coverage
- Branch Coverage
- Path Coverage
- Loop Testing
- Control Flow Testing

## Inti
"Apakah semua logika dalam kode sudah berjalan dengan benar?"
